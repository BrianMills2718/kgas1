# Summary

`carter_analysis_output` is a concrete generated-output slice from the lit-review experiment. It contains six files: a summary, two JSON analyses, two YAML schema/output files, and one integrated multi-theory JSON analysis. [1]

The outputs apply two academic theories to the same 20,636-character Carter speech:

- Young (1996), cognitive mapping meets semantic networks. [2][3]
- Chong & Druckman (2007), framing theory. [4][5]

The integrated output combines the two under a "Multi-paradigm balanced analysis" approach. [6]

# Output Structure

Both single-theory JSON outputs use the same five-purpose structure:

- descriptive analysis
- explanatory analysis
- predictive analysis
- causal analysis
- intervention analysis [2][4]

Both report `model_type: property_graph`, `analysis_type: balanced_multi_purpose`, `purpose_balance_score: 1.0`, no detected causal over-emphasis, and high theoretical fidelity. [2][4]

# Integrated Analysis

The integrated JSON says it combines:

- Carter as cognitive mapper plus legislative audience as frame processors
- peace-oriented beliefs plus strategic framing of US-Soviet relations
- formal speech, semantic networks, and psychological mechanisms
- Cold War competition plus post-Vietnam credibility crisis [6]

It maps the same five analytical purposes into combined cognitive-mapping and framing-theory effects. [6]

# Caution

This slice is stronger than a planning document because it is an actual generated output artifact. It is still internal evidence: the balance score, theoretical fidelity, and no-over-emphasis claims are generated or system-produced claims, not an independent validation.
