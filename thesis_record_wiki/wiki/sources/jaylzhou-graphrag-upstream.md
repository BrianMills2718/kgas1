# Summary

JayLZhou/GraphRAG is an external GraphRAG repository whose README identifies DIGIMON as "Deep Analysis of Graph-Based Retrieval-Augmented Generation (RAG) Systems." It frames the project as modularizing and decoupling graph-based RAG methods, and lists runnable method configurations such as RAPTOR, GraphRAG, LightRAG, HippoRAG, KGP, ToG, and others. [1]

The README also describes graph-type distinctions such as chunk trees, passage graphs, knowledge graphs, textual knowledge graphs, and rich knowledge graphs. It further decomposes retrieval into operator categories for entities, relationships, chunks, subgraphs, and communities. [1]

Brian's Digimon/KGAS work should be treated as an extension, fork, or local research line connected to this upstream/reference project, not as an isolated origin.
