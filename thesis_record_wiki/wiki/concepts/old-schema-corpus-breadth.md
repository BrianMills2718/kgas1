# Summary

Old schema corpus breadth is the finding that the lit-review experiment had already attempted a wide theory-schema corpus before later pipeline refinements. The preserved `old_schemas/` directory has 56 YAML files spanning multiple social-science, influence, persuasion, behavior-change, intervention, argumentation, and information-disorder domains. [1]

# Why It Matters

This breadth is historically important: the project was not only building KGAS infrastructure, but also trying to encode a large interdisciplinary theory library.

It also creates a validation burden. A broad schema corpus is not a validated schema corpus. Later work on model-form routing, prompt calibration, formal notation extraction, and universal application should be read as responses to this burden.
